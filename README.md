# User Catalog App

A simple and responsive user directory built with React and TypeScript, using the DummyJSON API.

## Features
- **User Listing**: Displays a clean grid of user profiles.
- **Search**: Real-time search by name (debounced for performance).
- **Pagination**: Easy navigation through the user list.
- **Responsive Design**: Works on mobile, tablet, and desktop.
- **TypeScript**: Fully typed for better developer experience and reliability.

## Tech Stack
- **Framework**: React 19
- **Language**: TypeScript
- **Tooling**: Vite
- **Icons**: Lucide React
- **API Client**: Axios
- **Styling**: Vanilla CSS

## Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. Clone the repository (if applicable)
2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App
Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`.

### Building for Production
To create a production build:
```bash
npm run build
```
The output will be in the `dist` folder.

# Приложение «Каталог пользователей»

Простой и адаптивный каталог пользователей, созданный с использованием React и TypeScript и работающий с API DummyJSON.

## Возможности
- **Список пользователей**: отображает аккуратную сетку профилей пользователей.
- **Поиск**: поиск по имени в реальном времени (с debounce для производительности).
- **Пагинация**: удобная навигация по списку пользователей.
- **Адаптивный дизайн**: работает на мобильных устройствах, планшетах и компьютерах.
- **TypeScript**: полная типизация для лучшего опыта разработки и надежности.

## Технологический стек
- **Фреймворк**: React 19
- **Язык**: TypeScript
- **Инструменты**: Vite
- **Иконки**: Lucide React
- **HTTP-клиент**: Axios
- **Стилизация**: Vanilla CSS

## Начало работы

### Предварительные требования
- Node.js (v18 или выше)
- npm или yarn

### Установка
1. Клонируйте репозиторий (если применимо)
2. Установите зависимости:
   ```bash
   npm install
   ```

### Запуск приложения
Запустите сервер разработки:
```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:5173`.

### Сборка для продакшена
Чтобы создать production-сборку:
```bash
npm run build
```
Результат будет в папке `dist`.
